About

Saya adalah mahasiswa di Universitas Palangkaraya tahun angkatan 2021, saya mengikuti program Studi Independen di Infinite Learning web development beberapa hal yang membuat saya tertarik untuk memilih web dev diantaranya adalah :

Alasan 1

Kemampuan untuk Mewujudkan Ide Kreatif: Mata kuliah web development memungkinkan saya untuk mengubah ide-ide kreatif menjadi kenyataan. Dengan memahami dasar-dasar pengembangan web, saya dapat membuat situs web dan aplikasi web sesuai dengan visi saya sendiri. yang mana akan memberi saya kendali penuh atas proyek-proyek online, portofolio pribadi, proyek bisnis, blog, atau aplikasi web lainnya. Kemampuan untuk mengembangkan situs web sendiri juga dapat menghemat uang yang mungkin akan saya habiskan untuk menyewa pengembang web.

Alasan 2

Permintaan Tinggi di Pasar Kerja: Saat ini, kehadiran online menjadi semakin penting bagi bisnis, organisasi, dan individu. Permintaan untuk pengembang web yang terampil terus meningkat karena banyaknya keperluan yang berjalan secara online. Dengan menguasai keterampilan web development, saya akan memiliki peluang yang lebih baik untuk mendapatkan pekerjaan yang stabil dan berkembang dalam industri teknologi informasi. Banyak perusahaan mencari pengembang web untuk membangun dan merawat situs web, yang membuat pengembang web menjadi aset berharga dalam pasar kerja saat ini.
